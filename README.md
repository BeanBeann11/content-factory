# 🏭 Content Factory

## Что здесь
- Мои чек-листы (готовые PDF)
- Промпты для AI
- Настройки для продаж (Robokassa + Tilda)

## Как работаю
1. Придумываю тему
2. Генерирую чек-лист через ChatGPT
3. Делаю обложку (Canva / Midjourney)
4. Загружаю в Robokassa
5. Публикую ссылку

## Инструменты
- ChatGPT / Gemini — текст
- Canva / Midjourney — обложки
- Robokassa — приём платежей + выдача PDF
- Tilda — страница продажи
- Telegram — трафик
