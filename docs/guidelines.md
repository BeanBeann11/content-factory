# Гайдлайны

## Создание чек-листа
- Объём: 5–15 пунктов
- Язык: простой, без воды
- Формат: маркированный список

## Дизайн обложки
- Размер: 1200×1600 px
- Стиль: минимализм, 2–3 цвета
- Инструмент: Canva (бесплатно)

## Загрузка в Robokassa
1. Зарегистрироваться
2. Добавить "цифровой товар" → загрузить PDF
3. Получить ссылку на оплату
4. Вставить в кнопку на Tilda
